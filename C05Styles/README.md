# Styles of C API
There are 4 styles:
 - Flat C APIs: FREE FUNCTIONS along with supporting data structures & constants. NO objects, NO inheritance.
 - Object-Oriented C++ APIs
 - Template-Based APIs: Template versus Macros
 - Data-Driven APIs

## Complementary Terms:
```CPP
new<>delete
malloc<>free // alloc??
create<>destroy
insert<>delete
source<>target
send<>receive
add<>remove
failure<>success
...
```
